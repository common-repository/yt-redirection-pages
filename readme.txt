Allows you to open external links via redirection pages, inline popup or frames. Users first goes to a redirection page & then goes to destination. 

== Description ==

YT Redirection Pages offers various options to redirect external or certain internal links via beautiful redirection pages, inline window or frame. Rather than showing your users external links directly, redirect them using redirection pages or ask them to accept some terms before you redirect them.

Upon activation of the plugin, you'll have option to select page design, page option (redirect or accept terms) etc. You can also link only few links manually to show redirection pages using short codes.

There are several fields to edit note texts, copyright texts and edit other parts of the redirection page.

Currently there are three options to redirect external links. (1) Redirection pages (2) Inline popup (3) Frame.

There are three designs for redirection pages with option to make it terms page or redirection counter page.

== Installation ==

The installation of this plugin is very easy. Install the plugin using any of of the following method

e.g.

1. You can simply search the plugin from Plugins > Add New (Using word 'YT Redirection' OR
1a. Download plugin and go to Plugin > Add News > Upload Plugin to upload zip file or Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->YT Redirection screen to configure the plugin
4. Check site's post where where you have published any article with external link. It should be redirecting using redirection page.

Select appropriate page design (5 options) and redirection methods (counter or accept/reject) terms, change texts on the redirection page etc.. to customize.

== Frequently Asked Questions ==

= Do i have to manually convert each link to show redirection page? =

You can do it manually or globally. When you uncheck "No Redirect" it will enable manual mode where each link you can set using shortcode.

= What does it do basically. =

After you activate this plugin, all the external links on your site will redirected via a redirection page. Meaning, when someone clicks on any external link they'll see a page (hosted on your site) such as www.yourblog.com/redirects/http://www.google.com

Depending on configuration, it will redirect user to final destination page either by clicking on Yes button or redirect after few seconds.

= What are the applications of this plugins? =

There are various applications for this plugin. 

Example 1: If you're relying on affiliate revenue, you can warn your visitors that you may earn some commissions when they make purchase from destination site.
Example 2: You may want your users to agree some terms like "You're visiting external page, we're not responsible for whatever action you make there blah blah"
